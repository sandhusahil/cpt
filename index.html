CPT – THE CODE OF PERSONAL TRANSFORMATION

Updated Edition – 2025

Updated on: 1 Dec 2025

PILLAR 1: UNDERSTANDING MYSELF

Path 1: Self-Reflection – Regular reflection on thoughts & actions
Path 2: Self-Awareness – Know values, triggers, beliefs
Path 3: Personality Exploration – Use tests/tools to know strengths & weaknesses

PILLAR 2: FITNESS

Path 1: Morning Routine – Wake 6:30 AM
Path 2: Exercise – Walk/run 30 mins daily
Path 3: Healthy Eating – Balanced diet
Path 4: Mental Fitness – Cognitive practice
Path 5: Rest – Sleep 7–8 hrs; bed 10:30–11:00 PM

PILLAR 3: KNOWLEDGE

Path 1: Lifelong Learning
Path 2: Critical Thinking
Path 3: Apply Knowledge in real life

PILLAR 4: ENTERTAINMENT

Path 1: Mindful Content – No harmful content
Path 2: Balanced Leisure – Work first, fun later
Path 3: Movies allowed; series optional

PILLAR 5: DIGITAL WELL-BEING (UPDATED)

Path 1: Content Quality – Only positive/educational; no harmful content
Path 2: YouTube Videos – After 12 PM; max 3 useful videos before 5 PM
Path 3: YouTube Shorts – Only after 5 PM
Path 4: Movies – Allowed 10 AM–10:30 PM
Path 5: Digital Boundaries – No phone first 30 mins; no random browsing; notifications off
Path 6: Weekend Flex – +1 hr entertainment; shorts still after 5 PM
Path 7: Bonus Growth – Write 1 line after any educational video

PILLAR 6: SPIRITUAL WELL-BEING

Path 1: Meditation & Prayer
Path 2: Introspection
Path 3: Spiritual Study
Path 4: Daily Gratitude

PILLAR 7: TIME MANAGEMENT

Path 1: Prioritize important tasks
Path 2: Reflect & adjust daily
Path 3: Follow a clear schedule

PILLAR 8: REVISION & ADAPTATION

Path 1: Revise only when needed
Path 2: Core pillars stay unchanged
Path 3: Reflect on alignment
Path 4: Update system in a structured way

PILLAR 9: ACCOUNTABILITY & ENFORCEMENT (NEW)

Path 1: Minor Violations
Includes: skipping walk, phone in first 30 mins, sleeping after 11, skipping meditation
Punishment: 5-min meditation + 10-min walk next day
Path 2: Major Violations
Includes: no study, no job application, YouTube before 12 PM, watching Shorts before 5 PM
Punishment: 1-hour No-Phone Zone (phone in another room)
Path 3: Extreme Violations
Includes: 2–4 hrs wasted on phone, skipped study + job, losing control
Punishment: Next day = 0 mins YouTube, no entertainment
Path 4: Rewards
7-Day Streak → Treat under ₹100
14-Day Streak → 1 movie
30-Day Streak → Buy something useful
Path 5: Daily Accountability
Every night ask:

Did I follow the CPT?
What did I break?
What punishment applies?
Path 6: Integrity Rule
If punishment is skipped → counts as Extreme Violation
Rewards only if all days fully followed
